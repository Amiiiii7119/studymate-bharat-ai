# StudyMate Bharat AI

StudyMate Bharat AI is an AI-powered learning and productivity assistant designed for Indian students preparing for school exams, competitive exams, and college-level technical subjects.

The project focuses on:
- Simplifying complex concepts using AI
- Personalized study planning based on syllabus and exam timelines
- Multilingual support (English and Hindi)
- Low-bandwidth and offline-first accessibility
- Developer and coding assistance for technical subjects

## Repository Contents
- `requirements.md` – Functional and non-functional requirements generated using Kiro
- `design.md` – System design and architecture generated using Kiro

## Purpose
This repository contains the specification and design artefacts submitted for the **AI for India Hackathon (Student Track)**.

This is a conceptual design focused on feasibility, scalability, and real-world impact.
